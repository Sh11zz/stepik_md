### ЦЕЛЬ 
ВОССТАНОВИТЬ ДОКУМЕНТАЦИЮ ПРОЕКТА ЦВЕТИК СЕМИЦВЕТИК

### ОПИСАНИЕ ПРОЕКТА
ЭТО ВЕБ ПРИЛОЖЕНИЕ ДЛЯ ЗАГАДЫВАНИЯ ЖЕЛАНИЙ

ИНТЕРФЕЙС НАПИСАН НА `HTML` `CSS` `JAVASCRIPT`
БЭКЕНД НА `NODE JS`

### КАК ЗАПУСТИТЬ
1. СКЛОНИРУЙТЕ РЕПОЗИТОРИЙ `git clone`
```
[Ссылка на репозиторий](https://github.com/tri-topora/cvetik-semicvetik)
```
2. ПЕРЕЙДИТЕ В ПАПКУ ПРОЕКТА
3. УСТАНОВИТЕ ЗАВИСИМОСТИ `npm install`
4. ЗАПУСТИТЕ СЕРВЕР `npm run start`
